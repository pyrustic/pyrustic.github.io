Dresscode : Dress up your code with a beautiful GUI

Dresscode is a high productivity framework for developing a graphical user interface without prior knowledge of using a GUI Toolkit.

As a high productivity framework, Dresscode is suitable for teaching, prototyping, testing, adding a GUI to command-line scripts, developing simple to complex desktop applications, etc.

Figure 1 - Demo built with 1 hex-digit lines of Python code

Dresscode comes with an innovative and original concept of pages automatically referenced in the menu bar.

This out-of-the-box user experience feature allows the software developer to focus on the creative side of programming and the user to enjoy an intuitive and ergonomic interaction with the app.

A page is a scrollable view that you can populate with an unlimited number of components. The components come with various configuration options and placeholders for event handlers. Data is retrieved from components with a simple method call.

Pyrustic Framework : Develop, package, and publish desktop applications

If Dresscode is Python, Pyrustic Framework would be C. Pyrustic Framework is a lightweight framework to develop, package, and publish Python desktop applications.

Figure 2 - Jupitest - Graphical test runner

The framework comes with a bunch of useful widgets as well as a default optional dark theme dubbed tk-cyberpunk-theme. The dark theme is created with an innovative built-in theme/style mechanism to both showcase the theme mechanism and also to inspire people to create their own themes that they can share with the world if they wish.

Figure 3 - Hubway - Minimalist Github client

The graphical user interface is not the only topic that Pyrustic Framework targets. The framework covers various other aspects of software development, from gui-toolkit-compatible multithreading to event notification and more. There are even a libary to fetch resources with an implementation of conditional request and a smart responses caching system.
Hubway the minimalist Github client, uses this library to fetch resources as a good API citizen.

Figure 4 - Rustiql - Graphical SQL editor

Pyrustic Framework is shipped with Pyrustic Manager, a command line tool to create/initialize/manage a project (new or existing), build a distribution package (Wheel), publish the distribution package, etc.

Under the hood, Dresscode uses Pyrustic Framework and expose the objects related to it.

Hubstore : Connect users to apps

Github's Release feature provides the ability to link assets to a release.

Hubstore is the implementation of the innovative and original idea of using this feature to deliver a software distribution pipeline that connects users to applications.

Figure 5 - Hubstore

Basically, as a software producer, you issue the build and publish commands to make your app available to users who will simply type in Hubstore your Github username slash your app name to get the latest version of your application.

Figure 6 - The Pyrustic application distribution pipeline

Hubstore comes with some cool innovative features like the rollback feature which reinstalls a recent snapshot of an app when the latest update is buggy.

Theming : Give your application a visual personality

Customize the look of your app to improve the user experience. Pyrustic Framework comes with an elegant theme/style mechanism so you can create your own theme easily without learning a new DSL. You can also simply use an existing theme. Thanks to the flexibility of the theming system, you can override the elements of an existing theme without altering its source code.

A dark theme named tk-cyberpunk-theme is available and a light theme will be released soon.

tk-cyberpunk-theme

This is the default dark theme used by Hubstore and Dresscode.

Figure 7 - Demo app without and with tk-cyberpunk-theme

Pyrustic is shipped with tk-cyberpunk-theme, so you don't need to install it manually. You can also decide to don't use any theme at all.
